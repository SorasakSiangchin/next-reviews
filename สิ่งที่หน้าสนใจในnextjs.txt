ความรู้เพิ่มเติม
- Headless CMS จะแยกการทำงานของ frontend, backend, และฐานข้อมูล ออกจากกันเป็นส่วนๆ 
ซึ่งมีประโยชน์อย่างมาก เพราะช่วยให้การทำงานหลายอย่างรวดเร็วขึ้น

- Strapi คือโปรแกรม headless CMS ที่เป็นโอเพนซอร์ส ซึ่งนักพัฒนาสามารถนำไปใช้สร้าง API ด้วย Javascript ได้อย่างรวดเร็ว 
และยังเป็นบริการที่ใช้งานได้ฟรี

- npm audit คือ คำสั่งที่เอาไว้ตรวจสอบ security dependencies ที่สร้างผลลัพธ์ report ออกมาในรูปแบบ json
